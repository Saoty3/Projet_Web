Le Cycle de l'Eau

Le captage de l'eau

Le captage de l'eau se fait de différentes manières : en rivières, dans les nappes phréatiques ou même dans la mer selon les pays. Il est important de savoir que l'eau ne représente pas une réserve fixe, mais fait partie d’un cycle, qui ne dessert pas tout le monde de manière équitable. Un habitant européen, par exemple, aura plus facilement accès à l’eau et pourra en consommer davantage (environ 150 litres par jour).

Le traitement de l'eau potable

Une fois l'eau récoltée, elle doit être conforme aux normes de l'Union Européenne. Elle est donc testée et traitée contre les métaux lourds, les nitrates, les nitrites, la pollution et certaines maladies (comme la Covid-19). Une fois traitée, elle peut être envoyée dans un château d'eau où elle sera stockée.

Le stockage de l'eau

L'eau est stockée à une certaine hauteur, ce qui lui donne la pression nécessaire pour atteindre les habitations même éloignées du château d'eau.

Le traitement de l'eau usée

Une fois utilisée, l'eau est considérée comme polluée. Pour respecter les normes européennes, elle doit être traitée avant d’être rejetée. Elle est donc envoyée dans une autre usine, où elle est filtrée afin d’éliminer les polluants. Une fois purifiée, l’eau peut être relâchée. La pollution extraite peut ensuite être utilisée pour produire de l’énergie.
